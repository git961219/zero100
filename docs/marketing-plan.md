# Zero100 - 마케팅 채널 전략

> 택님 최종 확인 필요 (대외 배포 자료, 자율성 A3)

---

## 1. 타겟 유저 페르소나

| 페르소나 | 프로필 | 핵심 니즈 | 접근 채널 |
|---|---|---|---|
| **튜닝 매니아** | 20~30대 남성, ECU/배기/흡기 튜닝 경험, 차량 커뮤니티 활동 | "튜닝 전후 수치로 증명하고 싶다" | 보배드림, 클리앙 자동차, 인스타 |
| **트랙데이 유저** | 30~40대, 서킷 주행 경험, 랩타임 기록 관리 필요 | "VBOX 없이 랩타임 재고 싶다" | 서킷 커뮤니티, 카카오톡 단톡방 |
| **해외 퍼포먼스 유저** | 글로벌, Dragy 대안 탐색, 가성비 중시 | "Dragy $200 대신 저렴한 대안" | Reddit r/cars, YouTube |
| **자동차 유튜버/리뷰어** | 콘텐츠 제작자, 측정 데이터로 영상 퀄리티 향상 필요 | "영상에 넣을 신뢰성 있는 제로백 데이터" | YouTube, 인스타, 협찬 제안 |

## 2. 초기 사용자 확보 전략

### Phase 1: 시드 유저 (출시 전~출시 1개월)

| 전략 | 실행 방법 | 목표 |
|---|---|---|
| 자동차 커뮤니티 직접 게시 | 보배드림, 클리앙, 디시인사이드 자동차 갤러리에 개발기 + 베타 모집 | 베타 테스터 50명 |
| 카카오톡 서킷 단톡방 | 인제/용인/영암 서킷 주행 단톡방에 소개 | 트랙데이 유저 20명 |
| Reddit/XDA | r/Android, r/cars, r/dragracing에 소개글 | 해외 유저 100명 |

### Phase 2: 초기 성장 (출시 1~3개월)

| 전략 | 실행 방법 | 목표 |
|---|---|---|
| 유튜버 시딩 | GPS 모듈 + 앱 세트로 자동차 유튜버 5명에게 제공 | 리뷰 영상 5개 |
| Dragy 비교 콘텐츠 | Dragy vs Zero100 직접 비교 영상/글 제작 | 검색 유입 핵심 |
| 리뷰 유도 | 측정 3회 완료 시 리뷰 요청 팝업 | 별점 4.5+ 유지 |

### Phase 3: 확장 (3~6개월)

| 전략 | 실행 방법 | 목표 |
|---|---|---|
| 일본/중국 현지화 마케팅 | 일본 みんカラ, 중국 汽车之家 게시 | 다국어 유저 확보 |
| 서킷 이벤트 협찬 | 국내 트랙데이 이벤트에서 무료 측정 부스 | 오프라인 인지도 |
| ASO 최적화 | 키워드/스크린샷/설명문 A/B 테스트 | 자연 유입 2배 |

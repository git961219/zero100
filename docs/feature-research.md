## 4. 우선순위별 로드맵 제안

### Phase 1 — 즉시 구현 (S등급, 1~2주 단위)
1. **G-force 실시간 표시** — 가속도 센서 데이터 이미 수집중이면 게이지 UI만 추가
2. **복합 테스트 (0-100-0, 0-200-0)** — 가속 종료 후 바로 감속 측정 연결
3. **CSV/GPX 내보내기** — 기록 상세 화면에 "내보내기" 버튼
4. **차량 프로필 관리** — 차량명, 사진, 제원 저장 → 결과에 차량 태그
5. **런 비교** — 같은 테스트 유형의 2개 결과를 그래프 오버레이

### Phase 2 — 다음 릴리즈 (A등급, 1~2개월)
6. **가속중 예상 시간 표시** — 현재 가속률 기반 실시간 예측
7. **기울기 보정 + 유효성 검증** — GPS 고도 변화율로 경사 감지, 0.95% 초과시 경고
8. **커스텀 테스트 구간** — 사용자가 시작/끝 속도를 자유롭게 설정
9. **날씨/기온 자동 기록** — OpenWeather API 등으로 측정 시점 기상 메타데이터
10. **G-G 다이어그램** — 산점도 형태, 제동/가속/코너링 영역 시각화

### Phase 3 — 중기 (B등급, 3~6개월)
11. **리더보드/랭킹** — 차종별 글로벌 랭킹 (서버 구축 필요)
12. **HUD 모드** — 야간 윈드실드 반사용 좌우반전 고대비 UI
13. **영상 오버레이** — 측정 결과를 카메라 영상 위에 합성 (Dragy 대항마)
14. **랩타이머** — 서킷 모드 완전 신규 (GPS 지오펜스 기반)

### Phase 4 — 장기 (C/D등급)
- OBD2 연동, 트랙 DB, 3D 리플레이, 소셜, 이벤트 등

## 5. 특별 주목 기능 상세

### 5-1. G-force 측정/표시
- **현황**: 거의 모든 경쟁앱에 있는 기본 기능
- **구현**: 스마트폰 가속도 센서(IMU) 또는 외부 GPS 25Hz 데이터에서 속도 미분
- **표시 형태**: 실시간 숫자 (종가속도 Gx, 횡가속도 Gy), G-G 원형 다이어그램
- **참고**: Dragy Pro는 6축 IMU 내장으로 정밀도 향상

### 5-2. 영상 오버레이 (Dragy 킬러 기능)
- **Dragy**: 듀얼 카메라 + 속도/G-force/3D 맵 오버레이, GoPro/DJI 영상 임포트 후 자동 동기화
- **RaceChrono**: 100/120 FPS 녹화, PiP 합성, 커스텀 오버레이 레이아웃
- **난이도**: 높음 (영상 인코딩 파이프라인, 프레임 동기화, 렌더링 성능)
- **가치**: SNS 공유에 핵심, 바이럴 마케팅 효과

### 5-3. 리더보드/랭킹
- **Dragy**: godragy.com에서 차종별/테스트별 글로벌 리더보드, 100만+ 사용자 기반
- **RaceBox**: 트랙/드래그 별도 리더보드
- **필요 인프라**: 백엔드 서버, 결과 검증(GPS 무결성, slope 체크), 차종 분류 체계
- **사용자 효과**: 재방문율/리텐션 대폭 향상

### 5-4. 기울기 보정
- **Dragy**: 자동 slope 계산, +/-0.95% 초과 시 결과 "invalid" 마킹
- **RaceBox**: slope corrections 내장
- **구현**: GPS 고도 데이터로 시작-끝 고도차 / 거리 = 경사도(%)
- **중요성**: 동일 조건 비교를 위한 필수 기능, 리더보드 공정성에도 필수

### 5-5. 밀도고도 (Density Altitude)
- **RaceBox 독점적 기능**: 실시간 DA 값 표시
- **공식**: DA = f(기온, 기압, 습도) — 표준대기 대비 공기밀도 보정 고도
- **의미**: 같은 차량이라도 DA에 따라 가속 성능 5~10% 차이 가능
- **구현**: 기압센서(스마트폰 내장) + 기상 API → DA 계산

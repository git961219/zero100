## 1. 수익화 전략

### 1-1. 수익 모델 비교

| 모델 | 구조 | 예상 가격대 | 장점 | 단점 | 리스크 |
|---|---|---|---|---|---|
| **프리미엄 (원타임)** | 유료 앱 1회 구매 | $9.99~$19.99 | 단순, 사용자 저항 적음, 초기 매출 예측 용이 | 지속 수익 없음, 업데이트 동기 약화 | 무료 앱 대비 다운로드 급감 |
| **프리미엄+IAP** | 기본 무료 + 고급기능 인앱결제 | 무료 + $4.99~$14.99 | 진입장벽 낮음, 전환율 측정 가능 | 무료 기능 범위 설정 어려움 | Google Play 98% 매출이 이 모델 (검증됨) |
| **구독형** | 월/연 구독 | $2.99/월 or $19.99/년 | 지속 수익, LTV 극대화 | 니치 앱에 구독 피로감 큼 | 이탈률 높을 수 있음 (사용 빈도 낮은 앱) |
| **하드웨어 번들** | 앱+GPS모듈 세트 판매 | $79~$129 세트 | 높은 객단가, 완성형 솔루션 | 재고/물류/AS 리스크 | 초기 투자 필요, 반품 처리 |
| **광고형** | 무료+배너/리워드 광고 | 무료 | 최대 사용자 확보 | 퍼포먼스 앱에 광고 = 신뢰도 하락 | CPM 낮은 니치 시장, 수익 미미 |

### 1-2. 추천 모델: 프리미엄+IAP (Freemium)

**무료 제공 (Free Tier)**
- 내장 GPS 기반 0-100km/h 측정 (정확도 낮지만 체험 가능)
- 기본 속도 그래프
- 최근 5회 기록 저장

**유료 잠금 해제 (Pro Unlock, $9.99 원타임)**
- 외부 GPS 모듈 연결 (25Hz)
- 전 구간 측정 (60/100/150/200)
- 무제한 기록 저장 + 내보내기(CSV)
- 1-foot rollout 보정
- 기록 비교 오버레이

**근거**: Google Play 매출의 98%가 무료+IAP 모델에서 발생. 니치 앱의 구독 모델은 사용 빈도 대비 부담이 큼.

### 1-5. 광고 모델 판단

**부적합.** 이유:
- 퍼포먼스 측정 앱에서 광고는 신뢰도/전문성 이미지 훼손
- 니치 시장 CPM이 낮아 유의미한 수익 불가
- 측정 중 광고 노출은 UX 파괴

| 버전 | 시점 | 핵심 기능 | 목표 |

### v1.5 추가 기능 (리텐션 + 소셜)

| 기능 | 필요 리소스 | 효과 | 우선순위 |
|---|---|---|---|

| **차량 프로필** (차종/튜닝 스펙 등록) | 1~2주 | 데이터 분류, 차종별 비교 가능 | 중간 |
| **날씨/온도 자동 기록** | API 연동 1주 | 조건별 비교 (튜너에게 중요) | 중간 |
| **OBD-II 연동** (RPM, 부스트) | BT OBD 연동 2~3주 | 데이터 풍부화, 차별화 | 중간 |

### v2.0 추가 기능 (수익 확대 + 전문성)

| 기능 | 필요 리소스 | 효과 | 우선순위 |

| **영상 오버레이** (주행 영상 + 계기판 합성) | 비디오 처리 4주+ | 유튜버 핵심 도구, 강력한 바이럴 | 높음 |
| **iOS 버전** | Flutter 전환 or 네이티브 2~3개월 | 시장 2배 확대 | 높음 |
| **다중 GPS 지원** (RaceBox, Dragy 등) | 프로토콜 분석 2~3주 | 기존 기기 보유자 유입 | 중간 |
| **데이터 분석 대시보드** (웹) | 웹 개발 4주 | Pro 사용자 lock-in | 중간 |

## 3. 경쟁 차별화

### 3-1. Dragy 대비 비교

| 항목 | Zero100 | Dragy DRG70-C | Dragy Pro |
|---|---|---|---|
| **가격** | 앱 $9.99 + 모듈 별도 (번들 시 $79~99) | $150 | $209 |
| **GPS 갱신율** | 25Hz (u-blox M9) | 10Hz -> 25Hz (u-blox 10세대) | 25Hz |
| **IMU** | 없음 | 없음 | 6축 IMU 내장 |
| **연결 방식** | USB OTG (유선, 지연 없음) | Bluetooth (무선) | Bluetooth |
| **앱 생태계** | 직접 앱 (자유 커스텀) | Dragy 전용 앱 (폐쇄적) | Dragy + RaceChrono |
| **리더보드** | v1.5 예정 | 있음 (대규모) | 있음 |
| **배터리** | 불필요 (폰에서 전원) | 내장 (25시간) | 내장 |
| **휴대성** | GPS 동글만 (초소형) | 전용기기 별도 휴대 | 전용기기 별도 휴대 |
| **다국어** | 한/영/일/중 | 영어 중심 | 영어 중심 |

**Zero100 우위**: 가격 (1/15~1/20), USB 유선 안정성, 아시아 시장 다국어, 앱 자체 커스텀 자유도
**Dragy 우위**: 브랜드 인지도, 대규모 리더보드/커뮨니티, Bluetooth 무선 편의성, IMU(Pro)

### 3-2. 내장 GPS 앱 대비 비교

| 항목 | Zero100 (외부 GPS) | 내장 GPS 앱 (SpeedView 등) |
|---|---|---|
| **GPS 갱신율** | 25Hz | 1~5Hz (기기 의존) |
| **정확도** | +/- 0.03초 수준 | +/- 0.5~1.0초 |
| **일관성** | 높음 (전용 안테나) | 낮음 (기기/위치/케이스 의존) |
| **추가 장비** | USB GPS 동글 필요 | 불필요 |
| **가격** | $9.99 + 모듈비 | 무료~$5 |
| **신뢰성** | 측정 결과 공유/비교 가능 | "내장 GPS라 정확도 의문" |

**Zero100 우위**: 압도적 정확도/일관성, 신뢰할 수 있는 데이터
**내장 GPS 앱 우위**: 추가 장비 불필요, 진입장벽 제로

### 3-3. 핵심 차별화 포지셔닝

```
"전문가급 정확도(25Hz 외부 GPS)를 앱 가격($9.99)에"
```

| 차별화 축 | 설명 |
|---|---|
| **가격 파괴** | Dragy $150~209 vs Zero100 번들 $79~99 (앱만 $9.99) |
| **오픈 하드웨어** | u-blox M9는 범용 모듈 -> 사용자가 직접 구매 가능, 벤더 종속 없음 |
| **아시아 시장 특화** | 한/일/중 다국어, 아시아 튜닝 문화에 최적화 |
| **USB 유선 = 저지연** | BT 앱 vs USB 직결 -> 데이터 누락/지연 없음 |
| **EV 친화** | EV 가속 측정 수요 급증, 전기차 커뮤니티 공략 |

## 4. "지금 안 하면 나중에 비싸지는 것"

| 항목 | 이유 | 지금 비용 | 나중 비용 |
|---|---|---|---|
| **ASO 키워드 선점** | "제로백", "0-100 측정" 등 핵심 키워드를 경쟁자 전에 선점 | 앱 등록비 $25 | 후발 진입 시 광고비 수십배 |
| **리더보드 DB 구축** | 사용자 데이터가 쌓일수록 가치 증가 (네트워크 효과) | Firebase 무료 티어 | 나중에 데이터 없으면 Dragy 리더보드와 경쟁 불가 |
| **자동차 유튜버 관계** | 지금은 무료 제공으로 관계 형성 가능 | 모듈 1개 ($30) | 나중엔 광고비 ($500~$5000/건) |
| **EV 시장 타이밍** | EV 가속 자랑 문화가 지금 형성 중 | 앱 출시만으로 진입 | 1년 후 경쟁 앱 난립 예상 |
| **다국어 (일본/중국)** | 일본 JDM 튜닝 + 중국 EV 시장 = 영어권 외 최대 시장 | 이미 구현됨 | 나중에 현지화하면 6개월+ |

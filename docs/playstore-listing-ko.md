## 앱 이름
**Zero100 - GPS 제로백 측정기**
(28자)

## 간단한 설명
외장 GPS 25Hz로 정확한 제로백/구간가속/랩타임 측정. 1-foot rollout 보정 지원.
(46자)

## 상세 설명

### Zero100 - 고정밀 GPS 퍼포먼스 미터

외장 GPS 모듈(u-blox M9, 25Hz)을 활용한 고정밀 차량 성능 측정 앱입니다.
스마트폰 내장 GPS(1Hz)로는 불가능한 정밀 측정을 $200짜리 전용 장비 없이 실현합니다.

---

**[핵심 기능]**

| 기능 | 설명 |
|---|---|
| 제로백 측정 | 0-100, 0-200 km/h 가속 시간 측정 (1/100초 단위) |
| 구간 가속 | 60-100, 80-120, 100-200 등 자유 구간 설정 |
| 1-foot rollout 보정 | 드래그 레이스 표준 기준 적용 가능 |
| 실시간 속도 그래프 | 가속 과정을 시각적으로 확인 |
| 구간 랩타임 | 서킷/구간 주행 시간 측정 |
| 기록 관리 | 전체 측정 기록 저장, 비교, 공유 |

---

**[왜 외장 GPS인가?]**

스마트폰 내장 GPS는 1초에 1번(1Hz) 위치를 잡습니다.
외장 u-blox M9 모듈은 1초에 25번(25Hz) 위치를 잡습니다.

- 내장 GPS: 오차 +/- 0.5초 이상
- 외장 25Hz GPS: 오차 +/- 0.04초 수준

실제 드래그 레이스 측정 장비(VBOX, Dragy)와 동일한 원리입니다.

---

**[사용법]**

1. u-blox M9 GPS 모듈을 USB OTG로 스마트폰에 연결
2. Zero100 앱 실행, GPS 신호 안정화 대기 (약 30초)
3. 측정 모드 선택 (제로백 / 구간가속 / 랩타임)
4. 주행 시작 - 자동으로 측정 시작/종료
5. 결과 확인 및 저장

**[자주 묻는 질문]**

Q. 스마트폰 내장 GPS로도 사용 가능한가요?
A. 내장 GPS 모드도 지원하지만, 정밀 측정을 위해 외장 GPS를 강력히 권장합니다.

Q. Dragy와 비교하면?
A. Dragy는 전용 BT 하드웨어($200)가 필요합니다. Zero100은 범용 u-blox 모듈(약 $30~50)로 동등한 정밀도를 제공합니다.

Q. 1-foot rollout이 뭔가요?
A. NHRA 드래그 레이스 표준으로, 차량이 1피트(30.48cm) 전진한 시점부터 타이머를 시작하는 방식입니다. 론치 컨트롤 반응 시간을 제외하여 순수 가속 성능을 비교할 수 있습니다.

Q. 어떤 차에서 사용 가능한가요?
A. 모든 차량에서 사용 가능합니다. 자동차, 오토바이, 카트 등 GPS 신호가 잡히는 환경이면 측정됩니다.
